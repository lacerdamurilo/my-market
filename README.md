### Requisitos para rodar o projeto localmente:
- IDE Para Spring Boot (Recomendado: Intellij)
- Java 23
- Postgres versão 16
- Alterar senha do postgres para "pass" ou no arquivo application.yml (spring.datasource.password)
- Rodar o App.java :)
